# Meta Probability Calibration V2

V2 meta-model probability calibration is fitted only from inner out-of-fold
probabilities. The calibration rows are generated by models that did not train
on the corresponding observations.

## Supported Methods

The trainer evaluates:

- no probability calibration
- sigmoid/Platt calibration
- isotonic calibration when the out-of-fold sample size is sufficient

The selected method is chosen using inner validation probability diagnostics,
not final holdout performance.

## Diagnostics

Every calibration report records:

- selected method
- rows used for calibration
- Brier score
- log loss
- expected calibration error
- reliability curve bins
- calibration by candidate side
- calibration by market regime

## Sizing Gate

Probability-based sizing is approved only when the calibration sample is large
enough and Brier score, log loss, and expected calibration error are within the
configured limits.

A model can still produce diagnostic predictions when calibration is weak, but
it cannot be approved for probability-based sizing.
